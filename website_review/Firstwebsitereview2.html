To say a review on the internet is undeniable, is undeniable. It's all been the same HTML sites, the same copy-paste text, the same navigation, search bars bounced from year to year. The sheer amount of time it has taken for internet to change is frankly ridiculous. Just a couple of decade ago, people had to actually work to make respectable websites.

Nowadays, everyone seems to be following in each other's footsteps, some artists quite literally using other websites to create and host their own websites. Take Wix, GoDaddy, and Squarespace for example: they all make making websites too easy.

It is frankly ridiculous.

Unorigional websites of these days give the internet a 7.99999/10. There is still a lot of room for improvement. Make the internet 90's again!
